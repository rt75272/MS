## Program Overview
This repository contains coursework and projects completed as part of the **MS in Artificial Intelligence and Machine Learning** program at Colorado State University Global. The program provides a comprehensive foundation in computer science, software development, and advanced AI/ML techniques, culminating in a capstone project applying machine learning and neural networks to real-world problems.

### Skills & Topics Covered
- **Programming & Software Development**: Python, software engineering principles, algorithm design
- **Artificial Intelligence**: AI foundations, knowledge representation, search algorithms
- **Machine Learning**: Supervised/unsupervised learning, model evaluation, neural networks
- **Computer Vision**: Image processing, feature extraction, deep learning for vision tasks
- **Professional Development**: Technical leadership, ethical decision-making, project management
